# Квіти Перемоги — React версія

Це React-проєкт (Vite + TypeScript), створений на основі ваших HTML-сторінок:
- Головна
- Квіти
- Галерея (слайдер)
- Калейдоскоп (інтерактивний SVG)

Оригінальні HTML-файли збережені у папці `../legacy/`.

## Запуск

У терміналі:

```bash
cd flowers-react
npm install
npm run dev
```

Vite покаже локальну адресу, зазвичай `http://localhost:5173`.

## Роутинг

- `/` — Головна
- `/flowers` — Квіти
- `/gallery` — Галерея
- `/kaleidoscope` — Калейдоскоп

## Збірка

```bash
npm run build
npm run preview
```
